# StoryDeck GitHub Launch Kit

This file is a practical launch sheet for publishing StoryDeck on GitHub and sharing the first demo outside the repo.

## Repo profile

### Repository name

`storydeck`

### Short description

Template-driven H5 story pages from Markdown, JSON, or rough prompts.

### Longer pitch

StoryDeck turns content into mobile-first HTML story pages with reusable structure, theme presets, and standalone export. It is built for launches, explainers, case studies, creator pages, campaigns, and internal communications.

## Recommended demo path

For a short walkthrough video or animated GIF:

1. Open StoryDeck on the default `Indie Launch` preset.
2. Switch to `Case Study` to show a different narrative feel.
3. Switch to `Feature Brief` to show the more structured layout.
4. Change the theme on one pack to prove the content can be remixed.
5. Export the HTML file.

Keep the whole flow under 30 seconds.
